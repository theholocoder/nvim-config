# Nvim Config

This is my personal configuration for neovim 0.12 using `vim.pack` instead of `lazy.nvim` or `packer`.
It also tries to use the latest features of neovim, such as LSP improvements or autocompletion.

It is a work in progress and is not yet complete. It will never be complete.

## Requirements

- Neovim >0.12.0
- [Optional] Omarchy
- [Optional] Tmux
